Week 5: Extending JavaScript with jQuery & AJAX

jQuery Fundamentals

Why jQuery? Simplifies DOM manipulation and event handling.

Vanilla JavaScript jQuery Equivalent
document.querySelector('.class') $('.class')
element.addEventListener('click', fn) $(element).click(fn)
element.innerHTML = 'text' $(element).html('text')

Including jQuery:

<script src="https://code.jquery.com/jquery-3.6.0.min.js"></script>

Common jQuery Methods

  • Selectors: $('p'), $('#id'), $('.class')
  • Events: .click(), .hover(), .on()
  • Effects: .hide(), .show(), .fadeIn()
  • DOM Manipulation: .html(), .text(), .append() 
// Example: Click handler with fade effect
$('#myButton').click(function() {
    $('#message').fadeIn().html('Button clicked!');
});

AJAX Concepts

Asynchronous JavaScript And XML: Load data without page refresh.

jQuery AJAX Methods:

  1. $.ajax() - Full-featured method
  2. $.get() - Simplified GET request
  3. $.post() - Simplified POST request
  4. $.load() - Load HTML fragment

AJAX Implementation

// Basic AJAX request
$.ajax({
    url: 'api/data',
    method: 'GET',
    success: function(response) {
        $('#result').html(response);
    },
    error: function(xhr, status, error) {
        console.error(error);
    }
});

// Shorthand GET request
$.get('api/data', function(data) {
    console.log(data);
});

Response Formats:

  • JSON (most common)
  • HTML fragments
  • Plain text
