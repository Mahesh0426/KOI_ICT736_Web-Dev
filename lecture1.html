Lecture 1: Web Development Fundamentals

Course Overview

This course covers:

  • HTML, CSS, web forms, and validation
  • Data-driven websites and web standards
  • Server-side authentication and CMS evaluation
  • Search engine optimization (SEO)

Internet Fundamentals

Key developments:

  • ARPANET (1969) - Early packet-switching network
  • TCP/IP (1983) - Foundation of modern Internet
  • World Wide Web invented by Tim Berners-Lee (1989):
    • URLs, HTTP, HTML, and browsers

Client-Server Model

How web communication works:

  1. Client (browser) sends HTTP request
  2. Server processes request
  3. Server sends back HTTP response

Server types: Web, Database, Application, Mail servers

Static vs Dynamic Websites

Static Websites Dynamic Websites
Shows the same content to everyone Shows different content to different users
Content is fixed and rarely changes Content can change based on user interaction
Built with HTML, CSS, and JavaScript Built with server-side languages (PHP, Python, etc.)
Simple, fast, and easy to host Can be interactive and personalized
Example:
A personal portfolio website or a company brochure site.
These sites display the same information to every visitor and rarely change. 		Example:
Social media platforms like Facebook or Twitter, and sites like YouTube.
These sites show different content based on who is logged in, user actions, or search queries.

HTML Basics

Key concepts:

  • HTML5 is the current standard
  • Elements consist of tags and content
  • Semantic markup (header, nav, article, footer)
  • Non-semantic markup (div, span)
