Lecture 4: JavaScript Fundamentals & DOM

JavaScript Basics

Core concepts:

  • Variables: let, const, var
  • Data Types: String, Number, Boolean, Array, Object
  • Operators: Arithmetic, Comparison, Logical
  • Functions: Declarations vs Expressions

// Variable examples
let message = "Hello World";
const PI = 3.14;

// Function example
function greet(name) {
    return `Hello, ${name}!`;
}

DOM Manipulation

Document Object Model (DOM): Tree representation of HTML

Method Description Example
getElementById() Selects element by ID document.getElementById('header')
querySelector() Selects first matching element document.querySelector('.btn')
addEventListener() Attaches event handler button.addEventListener('click', handler)

Common DOM Operations

// Changing content
document.getElementById('demo').innerHTML = "New content";

// Changing styles
element.style.color = "blue";

// Creating elements
const newPara = document.createElement("p");
newPara.textContent = "This is new!";
document.body.appendChild(newPara);

// Event handling
document.getElementById("myBtn").addEventListener("click", function() {
    alert("Button clicked!");
});

JavaScript in HTML

Implementation methods:

  1. Inline: 
    <button onclick="alert('Clicked!')">Click me</button>
  2. Internal: 
    <script>
    // JavaScript code here
</script>
  3. External: 
    <script src="script.js"></script>
