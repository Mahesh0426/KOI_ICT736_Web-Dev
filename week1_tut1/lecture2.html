Lecture 2: Cascading Style Sheets (CSS)

What is CSS?

CSS (Cascading Style Sheets) controls the visual presentation of HTML documents.

  • Separates content (HTML) from design (CSS)
  • Uses selectors to target HTML elements
  • Follows cascade (priority rules) and inheritance

CSS Syntax

selector {
    property: value;
    /* Example */
    color: blue;
    font-size: 16px;
}

The foundational concepts of CSS:

  1. Selector (e.g., h1, .class, #id)
  2. Property (e.g., color, margin)
  3. Value (e.g., red, 20px)

CSS Selectors

Selector Type Example Targets
Element p All <p> tags
Class .highlight Elements with class="highlight"
ID #header Element with id="header"

Box Model

Every HTML element is a rectangular box with:

  • Content (text/images)
  • Padding (space inside border)
  • Border (edge around padding)
  • Margin (space outside border)
div {
    width: 300px;
    padding: 20px;
    border: 5px solid black;
    margin: 10px;
}

Types of CSS Syntax

  1. Inline CSS (within HTML tags): 
    <p style="color: red;">Text</p>
  2. Internal CSS (in <style> tag): 
    <style>
    p { color: blue; }
</style>
  3. External CSS (separate .css file): 
    <link rel="stylesheet" href="styles.css">
